Google Sheet Setup (~10 min, one time)

1. Sheet columns — row 1 must have exactly these headers:
Job Name  Contractor  Install Date  Status  Notes

2. Status values: Scheduled  In Build  Tentative  Complete  On Hold

3. Dates: MM/DD/YYYY — e.g. 06/15/2025

4. Publish: File → Share → Publish to web → Sheet1 → CSV → Publish
Your Sheet ID is the string between /d/ and /pub in that URL.

5. Edit this HTML file in Notepad. Find:
const SHEET_ID = '1R2LpMzNtjFM8K5-8w911WOxiB8zWyeST9v5BXFtN-2U';
Swap in your real ID and save.

6. TV: Open in Chrome → F11 fullscreen. Auto-refreshes every 5 minutes.
